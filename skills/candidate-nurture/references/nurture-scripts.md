# 保温话术库

> 按"场景 × 阶段"分场景的话术模板。话术不是死板模板，而是方向参考——实际发送时根据候选人情况微调语气和细节。

## 总原则

1. **催面评**（对面试官）：礼貌+明确+给压力点（候选人在等）。不要催得太凶，但要让面试官知道"这事拖不得"。
2. **碰候选人**（对候选人）：**给具体进展/预期，不要空问候**。"在吗？""最近怎样？"是废问候；"二面反馈在汇总，预计X号给答复"才是有效保温。
3. **长期停滞**：先判断"还要不要这个候选人"——如果业务已经不积极，提示用户考虑终止，别浪费保温成本。

---

## 场景1：催面评（对面试官）

### 面试后 1-2 天（轻催）
```
X总，{候选人名}的{轮次}面试（{日期}）方便写下面评吗？候选人在等结果，我这边好推进下一步～
```

### 面试后 3-4 天（正常催）
```
X总，{候选人名}一面过去{N}天了，候选人那边一直在等结果，担心拖久了他有其他选择。
方便今天抽空写下评价吗？如果太忙，语音说两句我帮你记录也行～
```

### 面试后 5 天+（强催）
```
X总，{候选人名}的{轮次}面评还没提交，已经{N}天了。候选人那边我有点压不住了，他问了好几次。
这个候选人之前评价不错，怕拖久流失。能否尽快？或者您觉得有什么顾虑直接说，我判断要不要继续推进。
```

**催面评的判断逻辑**：
- 面评全文现在可取（`interview_records/attachments` 接口）。催之前可以先查一下——如果面试官其实写了但没点"提交"，能拿到内容就不算"没写"
- 如果连续催 2 次面试官都不交，提示用户"这个面试官可能对面评流程不配合，建议找他的leader沟通"

## 场景2：停滞在"等约面"（对候选人）

### 停滞 3-5 天
```
{候选人名}你好，关于{岗位名}岗位，这边还在协调面试官的时间，预计本周内会安排面试。
你那边最近一周哪几天比较方便？我先记下来协调的时候优先匹配～
```

### 停滞 5-7 天
```
{候选人名}你好，关于{岗位名}的面试安排，面试官最近出差/项目忙，时间还没敲定，预计下周初能定下来。
你那边看机会的节奏有变化吗？如果还在看，我这边抓紧推进～
```

### 停滞 7 天+
```
{候选人名}你好，之前聊的{岗位名}岗位还在推进中，不过面试官时间协调得比较久，抱歉让你等了。
想确认一下你目前的求职状态——还在看机会吗？如果有其他offer在手上也告诉我，我帮忙协调加急～
```

---

## 场景3：面试后等结果（对候选人）

### 面试后 2-3 天
```
{候选人名}你好，{轮次}面的反馈这边在汇总中，面试官在写评价，预计{具体日期}前给你答复。
感谢耐心等待～有进展我第一时间同步你。
```

### 面试后 4-5 天（面试官交了面评但还在走流程）
```
{候选人名}你好，{轮次}面已经有初步反馈了，整体评价不错，这边在走内部流程确认下一步。
预计{日期}前给你明确答复，保持联系～
```

---

## 场景4：面评通过，推进下一轮（对候选人·好消息）

```
{候选人名}你好，好消息🎉 {轮次}面通过了！
这边在安排下一轮（{下一轮类型}），预计{时间范围}，具体时间我协调好通知你。
你那边最近有什么不方便的时间段吗？我先避开～
```

---

## 场景5：面评不通过（对候选人·坏消息）

> ⚠️ 不通过消息**必须先跟用户确认措辞**，不能自动发。以下只是参考。

```
{候选人名}你好，关于{岗位名}的面试结果同步一下——经过综合评估，目前可能不是最合适的时机。
感谢你花时间参与面试，{可选：某个具体的正面反馈}。
如果以后有合适的岗位，我们再联系，祝你求职顺利～
```

## 场景6：Offer 阶段保温（候选人已拿offer，等入职）

### 入职前 D-7
```
{候选人名}你好，下周{X}就要入职了，提前确认一下：
1、入职材料（身份证/学历/银行卡）都准备好了吗？
2、入职当天上午9点到{地点}，联系人{HR姓名}
有疑问随时问我，期待你加入～
```

### 入职前 D-3
```
{候选人名}你好，入职倒计时3天📅 万事俱备了吗？
提醒一下带齐材料，周三上午见～
```

### 入职前 D-1
```
{候选人名}你好，明天就入职啦！明天上午9点{地点}见，{HR姓名}会接待你。
今晚好好休息，明天见～
```

## 场景7：节日/关键节点问候（批量保温）

> 适用于在管道里的候选人，节日主动问候 = 存在感。

```
{候选人名}你好，{节日}快乐🎉 
{岗位名}岗位这边还在推进中，节后给你同步进展。
假期愉快～
```

**节点清单**（自动保温的触发节日）：
- 春节、国庆、五一、中秋、端午（法定假）
- 圣诞、元旦（海外候选人更在意）
- 不建议：双11、情人节（太生活化，不像招聘方该发的）

---

## 话术使用的注意事项

1. **不要群发感**：话术里的 `{候选人名}` `{岗位名}` 必须替换成真实信息，让对方感觉是"单独给我发的"
2. **给具体预期**：避免"尽快""近期"这种模糊词，尽量给具体日期（"预计7-4前"）
3. **Emoji 适度**：技术岗少用（1个以内），运营/设计岗可以多用。不要每条都带🎉
4. **渠道选择**：飞书IM为主，电话用于"重要节点"（offer/不通过/长期停滞的紧急沟通）
5. **发送时机**：工作日 10:00-11:30、14:00-17:00 最佳。避免周一早（开会）和周五晚（下班）
